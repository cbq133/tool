⏱ 老化测试加速模型计算

Hallberg-Peck 模型 模型 1

用于计算稳态湿热严苛程度2的试验持续时间

AF = (RHtest / RHParking,Field)³ × exp[(Ea/k) × (1/(TParking,Field+273.15) - 1/(Ttest+273.15))]
ttest = tParking,Field / AF
AF 佩克模型加速因子
--
ttest 试验持续时间
--h
Lawson 模型（Rosen）模型 2

基于罗森模型的稳态湿热试验时间计算

AT/RH = exp[-(Ea/k) × (1/(Ttest+273.15) - 1/(TParking,Field+273.15)) + b × (RHtest² - RHParking,Field²)]
ttest = tParking,Field / AT/RH
AT/RH 罗森模型加速因子
--
ttest 试验持续时间
--h
⚠ 注：从 NIO-TP.20.001-2022 和 NIO-TP.20.001-2019 Spec 中的举例结果来看，试验时间仅增加19小时存在疑问，需确定。
Arrhenius 模型（标准）模型 1

基于阿伦尼乌斯模型的高温加速试验时间计算，支持多温度分布

AT,I = exp[(-Ea/k) × (1/(Ttest+273.15) - 1/(TField,I+273.15))]
ttest = top × Σ(pi / AT,I)
序号TField,I 温度分布 (°C)pi 工作时间百分比AT,I 加速因子pi / AT,I
ttest 试验时间
--h
Arrhenius 模型（偏差测试序列）模型 2

适用于高温性能下降的元器件，使用双温度测试序列

Ttest,I = IF(TField,I ≤ Topmax, Topmax, Tmax)
AT,I = exp[(-Ea/k) × (1/(Ttest,I+273.15) - 1/(TField,I+273.15))]
tTest,I = top × (pi / AT,I)
ttest = Σ tTest,I
序号TField,I 温度分布 (°C)Ttest,I 测试温度 (°C)pi 工作时间百分比AT,I 加速因子tTest,I 加速试验时间 (h)
ttest 试验时间
--h
⚠ 注：
  • 若 TField,I ≤ Topmax，则 Ttest = Topmax
  • 若 TField,I > Topmax，则 Ttest = Tmax
  • 试验中 t1 = 48h × topmax/ttest
  • 试验中 t2 = 48h × tmax/ttest
⚠ 注：
  • 通过修改绿色背景数据来计算试验时间
  • 温度分布如低于5个点，则在多余温度点下列 pi 输入0即可
Coffin-Manson 模型 温度循环

基于Coffin-Manson模型的温度循环耐久试验次数与时间计算

ΔTtest = Tmax - Tmin
ACM = (ΔTtest / ΔTField)C
NTempCyclesField = 2 × 365 × tlife
Ntest = NTempCyclesField / ACM
tcycle = 2 × (ΔTtest / Tgrad) + tSoak,min + tSoak,max
ACM Coffin-Manson 加速因子
--
Ntest 试验周期次数
--Cycles
tcycle 试验单次循环时间
--min
升温/降温时间
--min
总试验时间
--h
⚠ 注：ΔTtest = Tmax - Tmin；C 取值：无铅焊料 2.65，有铅焊料 2.5；Nfield 通常按10年每天2次循环估算。
